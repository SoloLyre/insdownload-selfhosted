# v0.1.0

First public self-hosted release.

## Highlights

- local macOS web UI
- profile and single-post task modes
- Instagram, Douyin, and Xiaohongshu support
- local task queue, logs, and result pages
- local browser-session-based download flows

## Notes

- macOS-first release
- self-hosted only
- local browser login is recommended for higher-quality and more complete results
- profile downloads fetch the whole account currently exposed to the local logged-in session

## Setup

```bash
./scripts/bootstrap_mac.sh
cp config.example.toml config.toml
./scripts/run_local.sh
```

Open `http://127.0.0.1:8123`.
